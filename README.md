# rawGallery

Password-gated photo gallery platform built on Cloudflare Workers + R2 + KV.

Designed for photographers who shoot RAW. Upload Sony ARW files directly — they're converted to JPEG client-side via LibRaw-Wasm, with originals preserved in R2 for download.

## Features

### Viewer

- **Password-gated access** — different passwords unlock different gallery spaces
- **Hierarchical organization** — Space > Gallery > Event > Photos
- **Preview thumbnails** — gallery and event cards show 3 random preview photos
- **Responsive photo grid** — auto-fills columns as the window widens
- **Adjustable zoom** — +/− controls to resize thumbnails, with `/` to reset to default; preference saved across sessions
- **Progressive loading** — photos render in batches of 20, with automatic load-more on scroll
- **Full-screen lightbox** — PhotoSwipe 5 with swipe, pinch-to-zoom, and keyboard navigation
- **Slideshow mode** — play/pause button in the lightbox auto-advances every 4 seconds
- **Download originals** — download button in the lightbox serves the original RAW file when available
- **3:4 portrait crops** — thumbnail grid uses portrait-oriented crops for visual consistency
- **Sorted views** — galleries sorted by title, events by date, photos by filename
- **Breadcrumb navigation** — shows Space Name / Gallery / Event with clickable links
- **Dark-first design** — photos look best against dark backgrounds
- **Mobile-first** — responsive grid, touch gestures, progressive loading

### Admin

- **Admin-only uploads** — viewers can browse and download, only admin can modify
- **Cascading dropdowns** — select Space → Gallery → Event with human-readable names
- **Summary counts** — dropdowns and stats show gallery/event/photo counts at a glance
- **Space password management** — view (with eye toggle), edit, and change viewer passwords per space
- **Create and delete** — create or delete spaces, galleries, and events with cascade (deleting a space removes all its galleries, events, photos, and R2 files)
- **Duplicate detection** — files already in the upload queue (same name and size) are auto-marked "Skip"
- **Drop zone gating** — upload area is greyed out until a valid Space/Gallery/Event target is selected
- **RAW file support** — Sony ARW, Canon CR2, Nikon NEF, DNG decoded client-side via LibRaw-Wasm in a Web Worker
- **Pre-generated thumbnails** — 400px thumbnails created client-side at upload time for instant grid loads
- **Date-prefixed event slugs** — event URLs include `YYYYMMDD-` prefix when a date is set

### Infrastructure

- **On-the-fly image resizing** — `@cf-wasm/photon` WASM resizing in the Worker with `?w=` parameter, cached at the edge via Cache API
- **Parallel metadata loading** — photo metadata fetched via `Promise.all` for fast list responses
- **Low cost** — Cloudflare Workers ($5/mo paid plan), R2 (free egress), KV

## Architecture

```
Browser (Alpine.js + Tailwind CSS via CDN)
  ↕ password auth / HMAC-signed session cookie
Cloudflare Worker (TypeScript)
  ↕ R2 (photos) + KV (metadata, sessions)
```

## Setup

### Quick start

```bash
cd worker && npm install
cd ..
./init.sh
```

`init.sh` walks you through setup interactively:
- Prompts for worker name, R2 bucket name, KV namespace name, site name, admin token (with sensible defaults)
- Creates the R2 bucket and KV namespace via wrangler
- Auto-generates an HMAC secret for session signing
- Writes `worker/wrangler.toml` (from `wrangler.toml.default` template)
- Writes `worker/.dev.vars` for local development
- Sets production secrets via `wrangler secret put`

### Deploy

```bash
./deploy.sh
```

### Create a space and upload

1. Visit `/upload.html` and enter your admin token
2. Use **+ New Space** to create a space with a viewer password
3. Use **+ New Gallery** and **+ New Event** to set up the hierarchy
4. Select the target Space / Gallery / Event from the cascading dropdowns
5. Drop photos (JPEG, PNG, HEIC, or RAW) into the upload area
6. Share the viewer password with guests — they log in at `/` and browse

## File Structure

```
site/                    Static frontend (served by Wrangler assets)
  index.html             Login page (with Admin link)
  gallery.html           Gallery browser (Alpine.js SPA)
  upload.html            Admin upload page
  css/theme.css          Dark-first design system (CSS variables)
  js/upload.js           Upload logic, RAW conversion, admin UI
  js/raw-worker.js       Web Worker for LibRaw-Wasm decode
worker/
  src/index.ts           Cloudflare Worker API (all endpoints)
  wrangler.toml.default  Template (checked into git)
  wrangler.toml          Generated by init.sh (gitignored)
  package.json
init.sh                  Interactive setup script
deploy.sh                Deploy to Cloudflare
```

## API

| Method | Path | Auth | Description |
|--------|------|------|-------------|
| POST | `/api/auth` | none | Validate password, set session cookie |
| POST | `/api/logout` | none | Clear session |
| GET | `/api/session` | cookie | Check session status (returns spaceName) |
| GET | `/api/galleries` | session | List galleries (with preview photos) |
| GET | `/api/galleries/:slug/events` | session | List events (with preview photos) |
| GET | `/api/galleries/:slug/events/:slug/photos` | session | List photos |
| GET | `/api/photo/:g/:e/:id` | session | Serve JPEG (optional `?w=` resize) |
| GET | `/api/photo/:g/:e/:id/thumb` | session | Serve thumbnail |
| GET | `/api/photo/:g/:e/:id/original` | session | Download original RAW |
| GET | `/api/admin/spaces` | admin | List spaces (with gallery counts) |
| POST | `/api/admin/spaces` | admin | Create space |
| DELETE | `/api/admin/spaces` | admin | Delete space (cascades) |
| PUT | `/api/admin/spaces/:id/password` | admin | Update space password |
| GET | `/api/admin/spaces/:id/galleries` | admin | List galleries (with event counts) |
| POST | `/api/admin/galleries` | admin | Create gallery |
| DELETE | `/api/admin/galleries` | admin | Delete gallery (cascades) |
| GET | `/api/admin/spaces/:id/galleries/:slug/events` | admin | List events (with photo counts) |
| POST | `/api/admin/events` | admin | Create event |
| DELETE | `/api/admin/events` | admin | Delete event (cascades) |
| PUT | `/api/admin/upload/:id/:variant` | admin | Upload file to R2 |
| POST | `/api/admin/photos/complete` | admin | Finalize photo metadata |
| DELETE | `/api/admin/photos` | admin | Delete photo |

## Tech Stack

- **Worker**: TypeScript, Cloudflare Workers
- **Image resize**: `@cf-wasm/photon` (WASM, Lanczos3 filter)
- **Frontend**: Alpine.js, Tailwind CSS (CDN), PhotoSwipe 5
- **RAW decode**: LibRaw-Wasm (browser Web Worker)
- **Storage**: Cloudflare R2 (photos), KV (metadata)
- **Plugins**: Alpine.js Intersect (progressive load-more)
